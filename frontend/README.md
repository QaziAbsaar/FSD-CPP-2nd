# Campus Hub - Frontend

This is the React frontend for Campus Hub, built with Vite, React Router, Axios, and Tailwind CSS.

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

The app will run on `http://localhost:3000`

## Build

```bash
npm run build
```
